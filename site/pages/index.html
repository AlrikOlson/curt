what makes it curt

  

    

      
Brevity is the architecture, not a style.

      
Every construct's o200k cost is measured before it enters the language. Equations and juxtaposition remove definition and call ceremony; untagged unions with full inference remove the ADT tax; a dense single-token stdlib does the rest. Human-readable formatting is a derived view (curt expand), not a property of the source.

      

        The language tour ▸
        The measurements ▸
      

    

    

      
Capability-gated effects — fs and net are deny-by-default; ungranted ops yield a rescuable err, never a crash.

      
Machine diagnostics — single-line JSON with a verbatim fix; agents repair in one turn.

      
Grammar shipped as constrained-decoding artifacts — syntax errors become impossible.

      
Embeds everywhere — out-of-process (MCP), portable (wasm), in-process (C FFI) — same diagnostics and capabilities across all three.
